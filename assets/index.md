# Asset Index

> Curated reference materials for the ADR skill.
> All assets listed here are high-value: directly actionable checklists,
> templates, or guidance. Start here when the skill is activated.
>
> Medium-value assets (strategic context, tool docs, alternative perspectives)
> are consolidated in [SUPPLEMENTAL.md](SUPPLEMENTAL.md).

> previously archived assets. Use `archive/` only when deeper context is needed
> beyond what the notes and supplemental summaries provide — it should not be
> loaded as primary context.
