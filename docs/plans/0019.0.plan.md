### Task 1.1: Add `rename` command to `nygard-agent-format.sh`    [medium]

**Description:** Add a `rename` subcommand to the nygard-agent format script
that renames an ADR file from its current slug to a new slug derived from a
new title. This is the convergence mechanism for the solve task — it
transitions `NNNN-tbd.md` to `NNNN-<decision-title>.md` and updates the
in-file heading to match.

**Implementation Notes:**

Add `cmd_rename()` function and wire it into the dispatch `case` block:

```bash
cmd_rename() {
  local dir num new_title slug old_file new_file display_number
  dir=$(resolve_dir)
  num=$(printf "%04d" "$1")
  new_title="$2"
  slug=$(slugify "$new_title")
  display_number=$(echo "$num" | sed 's/^0*//')

  # Find existing file
  old_file=""
  for f in "$dir"/${num}-*.md; do
    [ -f "$f" ] && old_file="$f" && break
  done

  if [ -z "$old_file" ]; then
    echo "ERROR: ADR $num not found in $dir" >&2
    exit 1
  fi

  new_file="$dir/${num}-${slug}.md"

  if [ "$old_file" = "$new_file" ]; then
    echo "No rename needed: $(basename "$old_file")"
    return
  fi

  if [ -f "$new_file" ]; then
    echo "ERROR: target file already exists: $new_file" >&2
    exit 1
  fi

  # Update heading (first line): # N. Old Title → # N. New Title
  sed -i '' "1s/.*/# ${display_number}. ${new_title}/" "$old_file"

  # Update Last Updated date
  local today
  today=$(adr_date)
  if grep -q '^Last Updated:' "$old_file"; then
    sed -i '' "s/^Last Updated:.*/Last Updated: ${today}/" "$old_file"
  fi

  mv "$old_file" "$new_file"
  echo "Renamed: $(basename "$old_file") → $(basename "$new_file")"
}
```

Update the dispatch block to include `rename)` and update the usage text.

**Test & Acceptance Criteria:**
- [x] `nygard-agent-format.sh rename <num> <new-title>` renames the file and updates the heading
- [x] Rename updates the `Last Updated` metadata field
- [x] Rename errors if the ADR number is not found
- [x] Rename errors if the target file already exists
- [x] Rename is a no-op (with message) if the slug is unchanged
- [x] Usage text includes `rename` subcommand

**Dependencies:** None

**ADR Reference:** ADR-0019, Decision §Interim Title Convention
