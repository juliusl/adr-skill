## Stage 1: Cache Functions

### Task 1.1: Add cache_work_item and lookup_work_item functions    [medium]

**Description:** Extend `work-item-adapters.sh` with two cache functions: `cache_work_item` (appends normalized JSON + `cached_at` to `.adr/var/work-items.jsonl`) and `lookup_work_item` (finds latest cached snapshot by remote+id). Ensure `.adr/var/` is created on first write.

**Implementation Notes:**

```bash
# cache_work_item — reads normalized JSON from stdin, appends with cached_at
cache_work_item() {
  local cache_file=".adr/var/work-items.jsonl"
  mkdir -p "$(dirname "$cache_file")"
  local now cached_at_ts
  now=$(date -u +%Y-%m-%dT%H:%M:%SZ)
  jq -c --arg ts "$now" '. + {cached_at: $ts}' >> "$cache_file"
}

# lookup_work_item — returns latest cached snapshot for remote+id
lookup_work_item() {
  local remote="$1" id="$2"
  local cache_file=".adr/var/work-items.jsonl"
  [ -f "$cache_file" ] || return 1
  grep "\"remote\":\"$remote\"" "$cache_file" | grep "\"id\":\"$id\"" | tail -1
}
```

**Test & Acceptance Criteria:**
- [x] `cache_work_item` appends to `.adr/var/work-items.jsonl`
- [x] Appended record includes `cached_at` timestamp
- [x] `lookup_work_item gh 42` returns the cached record
- [x] `lookup_work_item` returns latest when duplicates exist (tail -1)
- [x] `lookup_work_item` returns non-zero when not found
- [x] `.adr/var/` is created automatically if missing

**Dependencies:** None
