### Task 1.2: Create Decision struct with chosen_option    [small]

**Description:** Replace the `Section` type used for `decision` with a new `Decision` struct that has `chosen_option: Option<usize>`, `justification: Option<String>`, and `body: Option<String>` (legacy). Old files use `body`, new files use `chosen_option` + `justification`.

Note: `Decision` must NOT use `#[serde(deny_unknown_fields)]` тАФ it needs to accept both old (`body`) and new (`chosen_option` + `justification`) fields in the same struct. The `Adr` top-level struct keeps `deny_unknown_fields`.

**Implementation Notes:**
```rust
#[derive(Debug, Clone, Serialize, Deserialize, PartialEq)]
pub struct Decision {
    #[serde(default)]
    pub chosen_option: Option<usize>,
    #[serde(default)]
    pub justification: Option<String>,
    #[serde(default)]
    pub body: Option<String>,
}
```

**Test & Acceptance Criteria:**
- [ ] `Decision` struct exists with `chosen_option`, `justification`, and `body` fields
- [ ] `Decision` does NOT have `deny_unknown_fields`
- [ ] `Adr` struct uses `Decision` type for `decision` field instead of `Section`
- [ ] Old-format TOML with `decision.body` still parses
- [ ] New-format TOML with `decision.chosen_option` + `decision.justification` parses
- [ ] Round-trip works for both formats

**Dependencies:** None

**ADR Reference:** ADR-0057, Design rule 2

### Task 2.2: Update cmd_export for new Decision struct    [small]

**Description:** Update `cmd_export()` in `main.rs` to handle the new `Decision` struct. When `chosen_option` is present, resolve it to the option name and render as "Chose: [option name]" followed by the justification. When `body` is present (legacy), render it as before.

**Implementation Notes:**
```rust
// New format
if let Some(idx) = adr.decision.chosen_option {
    if let Some(opt) = adr.options.get(idx) {
        println!("Chose **{}** (Option {})", opt.name, idx + 1);
    }
    if let Some(ref j) = adr.decision.justification {
        println!("\n{}", j);
    }
}
// Legacy format
else if let Some(ref body) = adr.decision.body {
    println!("{}", body);
}
```
