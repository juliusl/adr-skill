## Stage 1: Template Infrastructure (ADR 24)

### Task 1.1: Update nygard-agent-format.sh template generation    [medium]

**Description:** Update the `generate_template()` function in `nygard-agent-format.sh` to emit the two new checkpoint sections. The Evaluation Checkpoint goes between Options and Decision. The Conclusion Checkpoint goes between Additional Quality Concerns and the `---` separator before Comments. Also update `cmd_init()` to include checkpoint sections in the bootstrap ADR for format consistency.

**Files to update:**
- `src/skills/author-adr/scripts/nygard-agent-format.sh`

**Implementation Notes:**
The Evaluation Checkpoint section:
```markdown
## Evaluation Checkpoint (Optional)
<!-- Gate: Options → Decision. Agent assesses and recommends. -->

**Assessment:** [Proceed | Pause for validation | Skipped — <rationale>]

- [ ] All options evaluated at comparable depth
- [ ] Decision drivers are defined and referenced in option analysis
- [ ] No unacknowledged experimentation gaps (ADR-0022 tolerance check)

**Validation needs:**
```

The Conclusion Checkpoint section:
```markdown
## Conclusion Checkpoint (Optional)
<!-- Gate: Quality Strategy → Review. Verify before requesting review. -->

**Assessment:** [Ready for review | Needs work | Skipped — <rationale>]

- [ ] Decision justified (Y-statement or equivalent)
- [ ] Consequences include positive, negative, and neutral outcomes
- [ ] Quality Strategy reviewed — relevant items checked, irrelevant struck through
- [ ] Links to related ADRs populated

**Pre-review notes:**
```
