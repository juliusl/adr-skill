## Stage 1: Adapter Scripts

### Task 1.1: Create work-item-adapters.sh with normalized schema and adapter functions    [medium]

**Description:** Create `src/skills/author-adr/scripts/work-item-adapters.sh` containing the normalized work item data model validation function and three adapter functions: `gh_adapter`, `ado_adapter`, and `local_adapter`. The script is sourced by format scripts and the caching layer — not invoked directly by users.

**Implementation Notes:**

The script provides:
1. `validate_work_item` — validates a JSON object against the normalized schema using jq
2. `gh_adapter` — transforms GitHub issue JSON (from MCP `github-mcp-server-issue_read`) to normalized model
3. `ado_adapter` — transforms ADO work item JSON (from MCP `ado-wit_get_work_item`) to normalized model
4. `local_adapter` — generates a normalized work item from user-provided title, with auto-generated ID

Key transformations:
- GitHub: `number` → `id`, `state` → `state` (open/closed only), labels array checked for `bug` to set type
- ADO: `System.Id` → `id`, `System.WorkItemType` → normalized type, `System.State` → normalized state
- Local: timestamp hash → `id`, user-provided title, state defaults to `open`

Each adapter outputs a single-line JSON object to stdout.

**Test & Acceptance Criteria:**
- [x] `work-item-adapters.sh` exists and can be sourced without errors
- [x] `gh_adapter` produces valid normalized JSON from GitHub-style input
- [x] `ado_adapter` produces valid normalized JSON from ADO-style input
- [x] `local_adapter` produces valid normalized JSON with auto-generated id
- [x] `validate_work_item` returns 0 for valid JSON, non-zero for invalid
- [x] Type mapping: GitHub `bug` label → `type: "bug"`, default → `type: "issue"`
- [x] State mapping: GitHub `open` → `open`, `closed` → `closed`; ADO `New` → `open`, `Active` → `active`, `Resolved` → `resolved`, `Closed` → `closed`

**Dependencies:** None

**ADR Reference:** ADR-0035, Decision §Normalized Work Item Schema, §Adapter Interface, §Type Mapping, §State Mapping
