### Task 1.2: Add `build.rs` compile-time guard [small]

**Description:** Create `src/crates/adr-skills/build.rs` that verifies `src/skills/` and `src/agents/` exist relative to the workspace root at compile time. Emit a panic if either is missing or empty. Walk all files in both directories and emit `cargo::rerun-if-changed` for each file (not just the directory) so that content changes trigger rebuilds. Also emit the git commit SHA as a compile-time env var.

**Implementation Notes:**
```rust
// build.rs
fn main() {
    let manifest_dir = std::env::var("CARGO_MANIFEST_DIR").unwrap();
    let workspace_root = std::path::Path::new(&manifest_dir).parent().unwrap().parent().unwrap();
    
    let skills_dir = workspace_root.join("src/skills");
    let agents_dir = workspace_root.join("src/agents");
    
    if !skills_dir.exists() || skills_dir.read_dir().map(|mut d| d.next().is_none()).unwrap_or(true) {
        panic!("src/skills/ is missing or empty — RustEmbed requires skill files at compile time");
    }
    if !agents_dir.exists() || agents_dir.read_dir().map(|mut d| d.next().is_none()).unwrap_or(true) {
        panic!("src/agents/ is missing or empty — RustEmbed requires agent files at compile time");
    }

    // Re-run on any file change inside embedded directories
    for dir in [&skills_dir, &agents_dir] {
        for entry in walkdir(dir) {
            println!("cargo::rerun-if-changed={}", entry.display());
        }
    }

    // Embed git commit SHA
    let output = std::process::Command::new("git")
        .args(["rev-parse", "--short", "HEAD"])
        .output();
    if let Ok(output) = output {
        let sha = String::from_utf8_lossy(&output.stdout).trim().to_string();
        println!("cargo::rustc-env=GIT_COMMIT_SHA={sha}");
    }
}

fn walkdir(dir: &std::path::Path) -> Vec<std::path::PathBuf> {
    let mut files = vec![];
    if let Ok(entries) = std::fs::read_dir(dir) {
        for entry in entries.flatten() {
            let path = entry.path();
            if path.is_dir() {
                files.extend(walkdir(&path));
            } else {
                files.push(path);
            }
        }
    }
    files
}
```

### Task 2.2: Define clap CLI with subcommands [medium]

**Description:** Create `src/crates/adr-skills/src/main.rs` with the clap CLI. Top-level args: `--prefix` (optional, overrides default base path), `--version`. Subcommands: `install` (with sub-subcommands `skills`, `agents`, `all`), `init`, `setup`.

**Implementation Notes:**
```rust
use clap::{Parser, Subcommand};

#[derive(Parser)]
#[command(name = "adr-skills", version = concat!(env!("CARGO_PKG_VERSION"), " (", env!("GIT_COMMIT_SHA"), ")"), about = "Install and manage ADR skills and agents")]
struct Cli {
    /// Override the install base path (default: ~/.copilot)
    #[arg(long)]
    prefix: Option<std::path::PathBuf>,

    #[command(subcommand)]
    command: Commands,
}

#[derive(Subcommand)]
enum Commands {
    /// Install skills and/or agents
    Install {
        #[command(subcommand)]
        target: InstallTarget,
    },
    /// Bootstrap .adr/ directory in the current project
    Init,
    /// Full setup: install all + init (replaces make install-user)
    Setup,
}

#[derive(Subcommand)]
enum InstallTarget {
    /// Install skill definitions to ~/.copilot/skills/
    Skills,
    /// Install agent definitions to ~/.copilot/agents/
    Agents,
    /// Install both skills and agents
    All,
}
```

## Stage 3: Install and Init Logic

### Task 3.1: Implement install functions [medium]

**Description:** Create `src/crates/adr-skills/src/install.rs` with functions: `install_skills(prefix)`, `install_agents(prefix)`, `install_all(prefix)`. Each function: resolves the target directory from prefix (default `~/.copilot`), removes the old directory if it exists, iterates embedded assets, creates directories as needed, writes each file. Print what was installed. On failure, report what was written and what remains.

**Implementation Notes:**
- Default base: `dirs::home_dir().join(".copilot")`
- Skills target: `<base>/skills/<skill-name>/`
- Agents target: `<base>/agents/`
- Use `std::fs::create_dir_all` for directory creation
- Use `std::fs::remove_dir_all` before writing (matching Makefile rm + cp behavior)
- Print each installed path for visibility
